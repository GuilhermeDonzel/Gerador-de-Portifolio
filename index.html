Crie uma conta no GitHub
Acesse o GitHub e crie sua conta gratuita. Se já tiver conta, pule este passo.
github.com
3
Crie um repositório novo
Após entrar no GitHub, clique no botão + no canto superior direito → New repository. Dê um nome (ex: portfolio), deixe marcado como Public e clique em Create repository.
💡 Dica: use um nome simples como portfolio ou meu-portfolio. Ele vai aparecer na URL do seu site.
4
Faça upload do index.html
Dentro do repositório criado, clique em Add file → Upload files. Arraste o arquivo index.html para a área de upload e clique em Commit changes.
Arraste o index.html aqui ou clique para selecionar
⚠ Não renomeie o arquivo. Ele precisa ser enviado como index.html.
5
Ative o GitHub Pages
No repositório, clique em Settings (aba no topo) → no menu lateral clique em Pages. Em Branch, selecione main e clique em Save. Aguarde 1-2 minutos.

💡 Troque SEU-USUARIO e SEU-REPOSITORIO pelo seu usuário e nome do repositório no GitHub.
